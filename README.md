
# Pet Friends

## Descrição

Site construído para a ONG Pet Friends (@petfriends_animal - Instagram), de  forma voluntária, para auxiliar na divulgação dos animais disponíveis e suas características.

## Como foi feito

O projeto foi desenvolvido com integração entre IA e desenvolvimento tradicional.

O código base foi gerado em HTML, CSS e JavaScript com uso do Claude segundo direcionamento previamente estabelecido. O código, então, foi adaptado às as necessidades estéticas e funcionais da ONG de modo manual.

## Funcionalidades

Permite verificar os animais que estão aguardando adoção, conhecer mais sobre o trabalho da ONG, sobre as diferentes maneiras de ajudar e como entrar em contato para realizar a adoção.
